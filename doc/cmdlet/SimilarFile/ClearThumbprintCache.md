# Clear-ImageStoreThumbprintCache
Deletes all thumbprint cache data used by [Resolve-ImageStoreSimilarFiles](SimilarFile/ResolveSimilarFiles.md).

Alias: ClearThumbprintCache

# Parameters
None.

# Return
None.
